# /factory:compound

## Preflight

- Verify the problem has been solved or the implementation has been merged.
- Verify the relevant notes, files, and decisions are available.

## Plan

- Summarize the problem, the solution, and the key decisions.
- Capture any reusable patterns, risks, or follow-up work.
- Append durable learnings to the project knowledge loop.

## Commands

- Write the solution summary for `docs/solutions/`.
- Append the learning record to `.factory/learnings.jsonl`.

## Verification

- The new knowledge should be searchable and specific enough to help the next similar task.

## Summary

- Result: solution captured.
- Status: success only when the knowledge artifact is saved.

## Next Steps

- Return to the normal workflow on the next task.
